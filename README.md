# Titanic Survival Prediction - First ML Model

This is my first machine learning project using the Titanic dataset.

**Results**: Achieved 81.0% accuracy on test data  
**Key Finding**: Being male reduced survival odds the most with an importance score of -2.59. First class passengers had the highest survival rates.  

**How to run**: Click the blue `Open in Colab` button above → `Runtime` → `Run all`

Built with Python, Pandas, and Scikit-learn Logistic Regression.
